CS猥琐流脚本

Posted on July 15, 2009
Tags: Coding

CS猥琐流的精髓可以总结成五个字：“躲起来阴人”。一个合格的猥琐流选手必须具备以下两点修养：1. 走位风骚 (rushing)；2. 意识淫荡 (camping)。猥琐流打法常常被不明真相的群众污蔑为“蹲点/坑”。事实上猥琐流选手面对敌军避其锋芒，以逸待劳，二战著名狙击手灵魂附体，他不是一个人在战斗。

但凡成功必然要付出代价，要练出上面的技术，依现实或是虚拟世界，可能造成的运动伤害包括：膝关节损伤/痔疮或者腕管炎。

对此我也曾迷惘过，直到我研制出了持续前进/下蹲的CS脚本：

// Autoexec.cfg
alias +back__ "-forward;+back;reset_wheelupdown"
alias -back__ "-back;alias wheeldown ducking_T"

alias +forward__ "+forward;reset_wheelupdown"
alias -forward__ "-forward;alias wheeldown ducking_T"
alias goingForward_T "+forward;alias wheeldown goingForward_F"
alias goingForward_F "-forward;alias wheeldown ducking_T"
alias wheelup goingForward_T

alias +duck__ "+duck;reset_wheelupdown"
alias -duck__ "-duck;alias wheelup goingForward_T"
alias ducking_T "+duck;alias wheelup ducking_F"
alias ducking_F "-duck;alias wheelup goingForward_T"
alias wheeldown ducking_T

alias action_wheelup "wheelup;alias [count_wheelup] reset_wheelup"
alias action_wheeldowwn "wheeldown;alias [count_wheeldown] reset_wheeldown"
alias count0 "alias count count1;"
alias count1 "action"
alias reset_counter "alias count count0;alias [count_wheelup] count_wheelup;alias [count_wheeldown] count_wheeldown"
alias reset_wheelup "alias [count_wheelup] reset_counter"
alias reset_wheeldown "alias [count_wheeldown] reset_counter"
alias reset_wheelupdown "reset_wheelup;reset_wheeldown"
alias count_wheelup "count;alias action action_wheelup;reset_wheeldown"
alias count_wheeldown "count;alias action action_wheeldowwn;reset_wheelup"
reset_wheelupdown

// config.cfg
bind "w" "+forward__"
bind "s" "+back__"
bind "SHIFT" "+duck__"

bind "MWHEELDOWN" "[count_wheeldown]"
bind "MWHEELUP" "[count_wheelup]"

用CS（这里指Computer Science）黑话来说，这堆命令其实就是实现一个有限状态机（Finite-state machine）的功能，画个状态转移图就一目了然了。

使用方法：

  • 持续前进：
    • 鼠标滚轮向上连续滚动三次；
  • 停止持续前进：
    • 鼠标滚轮向下连续滚动三次，或
    • 松开W键，或
    • 按下S键
  • 保持下蹲姿势：
    • 鼠标滚轮向下连续滚动三次；
  • 起立：
    • 鼠标滚轮向上连续滚动三次，或者
    • 松开SHIFT键

有了这段脚本轻轻松松就能练就一身蛤蟆功，“行也蹲，站也蹲，卧也蹲”，因为不用一直按着SHIFT键，小拇指没有痉挛之虞，每次都可以躲起来用装了消音器的M4A1对着敌人（和友军）的后脑勺无情地扫射，真是太爽了。

P.S. 非作弊脚本，主要目的是保护手指。
