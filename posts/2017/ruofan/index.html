买了若饭的需要注意： 若饭v1.0咀嚼片版本在食品法规里属于“压片糖果”，该类食品不允许添加矿物质以及维生素（VC除外）等营养补充剂。 若饭v2.0粉状版本在食品法规里属于“固体饮料”，该类食品可添加除了维生素K和生物素以外的各种常见维生素，但是矿物质方面仅允许使用铁、钙、锌、镁和磷。 因此若饭营养成分表里有个别微量元素的NRV并不完美，因此我们始终建议若饭v1.0不要连续吃三顿以上，若饭v2.0不要连续吃15顿以上。传统食物还是有必要的。除非等我们强大到能参与食品法规修订的那一天。

推荐《不存在日报》的文章《吃了两个月的粉末食物后，我来和你聊聊人类欲望》之“B面：意外收获——‘代餐粉社会学’”写的真是非常有科幻感。

不要相信若饭可以长期吃的谎言。

Soylent 可以长期吃， 那是因为人家的营养成分表各项均衡。再看看若饭的，钾和硒低成这样，您真敢靠它维生吗？对此官方长期以来建议“吃五停二”，但理由却是扯淡的“现在科学还不够发达，可能有某些人体必需营养素尚未发现”（大意），你们连 NRV 都配不平衡，还要把账赖到科学头上？我知道原因是若饭只拿到【固体饮料】经营牌照，被勒令不得擅自添加各种微量元素，但是在官网能看到相关说明吗？（我只看到煽情软文《若饭v1停产，“不作恶”到底有多难》）

若饭老板在知乎回答里说若饭“额外的强化了一些过量摄入不会带来负面影响的营养素，并且还要考虑到营养吸收问题”。“营养吸收问题”是偷换概念，营养推荐表指的是建议摄入量，跟吸收有什么关系？如果人群普遍吸收率低那么推荐摄入量自然会提高。“过量摄入不会带来负面影响”也是自作聪明的做法，营养推荐表本来就有平均需要量、推荐摄入量、适宜摄入量和可耐受最高摄入量等等指标，充分考虑了个体差异，有严谨的统计模型，毕竟成为营养学家不像 git clone soylent 那么简单。

若饭还有个硬伤：它冲泡出的液体总有溶解不尽的颗粒感，这些小颗粒容易附着在牙齿上造成龋齿/牙结石，而且漱口不管用。如果牙齿坏了，那就真的只能终身吃若饭了。

对比若饭的替代品，比如 Soylent，Joylent，全安素，雀巢佳膳等，在原料上都完胜若饭，营养成分表就更不用提了。而若饭的价格也没有明显优势。
